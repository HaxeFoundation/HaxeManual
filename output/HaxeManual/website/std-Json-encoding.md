## 10.10.2 Encoding JSON

Use the [haxe.Json.stringify](http://api.haxe.org/haxe/Json.html#stringify) static method to encode a Haxe value into a **JSON** string:
```haxe
class Main {
  static function main() {
    var o = {rating: 5};
    var s = haxe.Json.stringify(o);
    trace(s); // {"rating":5}
  }
}

```
